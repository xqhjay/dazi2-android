## 1. 产品概述

### 1.1 产品定位
一款纯工具型中文打字速度练习 App。通过下落字机制训练用户中文输入速度与准确率，提供数据化反馈与个性化练习内容。去除一切游戏化/修仙主题包装，回归练习工具本质。

### 1.2 目标用户
- 希望提升中文拼音输入速度的移动端用户
- 学生、文字工作者、需要大量中文输入的职场人群
- 偏好极简工具风、重视数据反馈的用户

### 1.3 核心价值
- **专注练习**：三种练习模式覆盖限时/自由/无尽场景
- **数据驱动**：WPM、准确率、错字本、趋势图，量化进步
- **个性化**：自定义字集导入，针对薄弱字强化
- **离线可用**：纯本地存储，无网络依赖

### 1.4 与参考项目差异
| 维度 | zjxx（参考） | 打字提速（本项目） |
|---|---|---|
| 主题 | 修仙游戏化包装 | 纯工具，无游戏化 |
| 平台 | Web/PC | Android 移动端 |
| 输入 | PC 键盘 + IME | 软键盘拼音 IME |
| 数据 | localStorage 简单存储 | SQLite 结构化存储 + 统计分析 |
| 扩展 | 无 | 统计/成就/自定义字集/主题音效 |
| UI | 水墨古风 | 现代极简工具风 |

## 2. 用户故事

### US-01 启动与首页
**作为**一名用户，**我希望**打开 App 即看到练习入口和今日数据概览，**以便**快速开始练习并了解自己的练习状态。

### US-02 限时挑战
**作为**一名用户，**我希望**进行 4 分钟限时挑战，速度随等级递增，**以便**测试自己在压力下的极限速度。

### US-03 自由练习
**作为**一名用户，**我希望**自定义下落速度进行不限时练习，打错的字会智能重现，**以便**针对薄弱字反复训练。

### US-04 无尽加速
**作为**一名用户，**我希望**进行速度递增的无尽模式，**以便**挑战自己的持久力与极限。

### US-05 数据统计
**作为**一名用户，**我希望**查看 WPM、准确率、练习时长趋势图和错字本，**以便**量化自己的进步并定位薄弱点。

### US-06 成就与打卡
**作为**一名用户，**我希望**通过每日打卡和成就徽章获得持续练习的动力，**以便**养成练习习惯。

### US-07 自定义字集
**作为**一名用户，**我希望**导入自定义字集（生字、专业词汇、诗词），**以便**针对实际需要练习。

### US-08 个性化设置
**作为**一名用户，**我希望**切换主题、开关音效与振动，**以便**在不同场景下获得舒适体验。

### US-09 移动端输入体验
**作为**一名移动端用户，**我希望**输入框固定底部、IME 弹起时下落区自适应，**以便**输入时仍能看到下落字。

## 4. 详细功能规格

### F-01 / F-02 三大练习模式与游戏机制

#### 4.1.1 限时挑战模式
- **时长**：4 分钟（240 秒）
- **机制**：字从顶部下落，用户输入对应汉字消除
- **生命值**：初始 100，每个字触底扣 10，归零则挑战失败
- **速度等级**：1-8 级，速度倍率 0.5× → 4.0×（每级 +0.5×）
- **升级**：挑战成功（血量未归零）则等级 +1
- **失败**：不降级，可重新挑战
- **结果**：显示 WPM、准确率、消除字数、是否升级

#### 4.1.2 自由练习模式
- **时长**：不限时，显示本局时长
- **速度**：用户自定义 0.1–5.0 倍速（数字输入）
- **暂停**：支持暂停/继续（按钮，无键盘快捷键）
- **智能错字强化**：打错或漏掉的字，10 秒后高概率（约 70%）再次出现；连续正确 2 次后恢复正常概率
- **生命值**：无（自由练习不失败）
- **结果**：显示 WPM、准确率、练习时长、错字列表

#### 4.1.3 无尽加速模式
- **机制**：初始 0.5×，每 15 秒加速 0.1×
- **生命值**：初始 100，字触底扣 10，归零则结束
- **得分**：每消除一个字 +1 分，速度越快得分倍率越高
- **记录**：最高分自动保存
- **结果**：显示得分、是否破纪录、WPM、准确率、坚持时长

#### 4.1.4 通用游戏机制
- **游戏循环**：requestAnimationFrame，60fps
- **下落字**：随机从当前字集选取，避免连续重复
- **同时下落数**：移动端限制 1-3 个（视速度等级），避免小屏拥挤
- **输入处理**：
  - 监听 `compositionstart` / `compositionend` 兼容 IME
  - 输入框固定底部，每次输入完成后清空
  - 输入内容与屏幕上所有下落字匹配，命中则消除最早出现的那个
- **视觉反馈**：消除时字淡出 + 轻微缩放动画；错字时输入框边框闪红

### F-03 字集管理
- **内置字集**：
  - 常用字 500（第 1-500）
  - 常用字 501-1000
  - 常用字 1001-1500
- **切换**：设置页或练习模式选择页切换，各模式等级/最高分按字集独立
- **格式**：`{ "title": "字集名", "words": ["字", "一", ...] }`

### F-04 本地数据存储
- **技术**：tauri-plugin-sql（SQLite）
- **表结构**：
  - `practice_records`：单局记录（模式、字集、WPM、准确率、得分、时长、时间戳）
  - `char_mastery`：单字掌握度（字、字集、正确次数、错误次数、最后练习时间）
  - `settings`：键值对（主题、音效、振动、当前字集、各模式等级/最高分）
  - `achievements`：成就解锁记录
  - `checkins`：每日打卡记录
- **字集文件**：内置字集打包进 APK，自定义字集存于 app data 目录

### F-05 / F-08 数据统计
- **单局结果**：WPM、准确率、消除字数、错字数、得分/时长
- **统计页**：
  - 概览卡片：累计练习时长、总消除字数、平均 WPM、平均准确率
  - 趋势图：近 7 天 / 30 天 WPM 与准确率折线图
  - 错字本：按错误次数排序的字列表，可点击进入针对练习
- **WPM 计算**：正确输入字数 / 练习分钟数
- **准确率**：正确输入次数 / (正确 + 错误 + 漏字) × 100%

### F-09 成就与打卡
- **每日打卡**：每天首次完成一局练习自动打卡，记录连续天数
- **成就徽章**（示例）：
  - 初心者：完成首次练习
  - 勤勉之士：连续打卡 7 天
  - 百字斩：单局消除 100 字
  - 速度达人：单局 WPM ≥ 40
  - 精准之手：单局准确率 ≥ 98%
  - 千字大师：累计消除 1000 字
  - 坚持不懈：连续打卡 30 天

### F-10 自定义字集导入
- **入口**：字集页 → "导入字集"
- **方式**：选择本地 JSON 文件
- **校验**：文件格式、字数（≥10）、非空
- **存储**：复制到 app data 目录，注册到字集列表

### F-11 智能错字强化
- 见 4.1.2 自由练习机制

### F-06 移动端布局
- **导航**：底部 Tab（练习 / 统计 / 字集 / 我的）
- **练习页布局**：
  - 顶部：模式信息条（模式名、计时/得分、生命值）
  - 中部：下落字游戏区（自适应高度）
  - 底部：固定输入框 + 当前模式控制（暂停等）
- **IME 避让**：监听视口尺寸变化（visualViewport API），IME 弹起时压缩游戏区高度，保证输入框可见且下落区不被遮挡
- **安全区**：适配刘海屏/导航栏，使用 safe-area-inset
- **触控**：所有可点击元素 ≥ 44pt，按钮间距充足

### F-07 设置
- **主题**：明色 / 暗色 / 跟随系统
- **音效**：开关（消除音、错误音、升级音）
- **振动**：开关（消除、错误、升级时短振动）
- **字集**：切换默认字集
- **重置**：清除所有数据（二次确认）

## 6. UI/UX 关键要求

### 6.1 视觉风格
- **风格**：现代极简工具风（参考 Notion / Linear / Apple 原生应用）
- **配色**：
  - 明色：背景 #FAFAFA / 卡片 #FFFFFF / 主文 #1A1A1A / 次文 #6B7280 / 强调 #3B82F6
  - 暗色：背景 #0F0F0F / 卡片 #1A1A1A / 主文 #F5F5F5 / 次文 #9CA3AF / 强调 #60A5FA
- **字体**：系统默认无衬线（中文 PingFang/Source Han Sans），游戏区字号 ≥ 32sp
- **圆角**：卡片 12-16dp，按钮 8-12dp
- **间距**：8dp 网格系统
- **动效**：克制，仅用于状态反馈（消除淡出、按钮按压、页面切换）

### 6.2 交互原则
- **单手可达**：核心操作集中在屏幕下半部分
- **即时反馈**：输入、消除、错误均有视觉/触觉反馈
- **无干扰**：练习中无广告、无弹窗，仅必要信息
- **容错**：误触返回有二次确认（练习进行中）

## 8. 验收标准（关键）

- AC-01：三大模式均可正常开始、进行、结束并记录数据
- AC-02：软键盘拼音输入可正确消除下落字，IME 弹起不遮挡游戏区

- AC-04：统计页正确展示趋势图与错字本
- AC-05：每日打卡与成就解锁逻辑正确
- AC-06：自定义字集可导入并用于练习
- AC-07：明/暗主题切换正常，设置持久化
- AC-08：APK 可在 arm64-v8a 真机安装运行
- AC-09：杀进程后数据不丢失
